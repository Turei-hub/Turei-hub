## Automation

### PR Review and Monitoring
This repository includes an automated GitHub Actions workflow that monitors pull requests and generates comprehensive review reports. The workflow:
- Automatically analyzes all PR changes
- Provides detailed statistics (files changed, lines added/deleted)
- Posts review reports as PR comments
- Updates reports when PRs are synchronized

See [`.github/workflows/README.md`](.github/workflows/README.md) for more details.
